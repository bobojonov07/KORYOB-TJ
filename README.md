
# KORYOB.TJ — Платформаи №1 барои дарёфти ҷойҳои корӣ дар Тоҷикистон

Ин лоиҳа бо истифода аз Next.js ва Firebase сохта шудааст. Шумо метавонед онро дар **Vercel** ройгон сайт кунед.

## 🚀 Чӣ тавр версияи навро ба GitHub бор кунед (Update):

Агар фармони дуюм сурх шавад, хавотир нашавед. Ин 3 фармонро паиҳам ва бодиққат иҷро кунед:

```bash
git add .
git commit -m "Update: Fullscreen UI and notifications"
git push origin main
```

*Эзоҳ: Агар номи "main" кор накунад, "master"-ро санҷед.*

## 🌐 Чӣ тавр дар Vercel сайт кунед (Deployment):

1. Ба [vercel.com](https://vercel.com/) ворид шавед ва бо GitHub пайваст шавед.
2. Тугмаи **"Add New"** ва баъд **"Project"**-ро пахш кунед.
3. Репозиторияи `KORYOB-TJ`-ро интихоб кунед.
4. Дар бахши **Environment Variables**, ҳамаи калидҳои Firebase-ро аз файли `src/firebase/config.ts` илова кунед.
5. Тугмаи **"Deploy"**-ро пахш кунед.

### 🔗 Пайваст кардани домени шахсӣ:
1. Дар Vercel ба бахши **Settings** -> **Domains** гузаред.
2. Домени худро (масалан `koryob.tj`) илова кунед.
3. Танзимоти DNS-ро, ки Vercel медиҳад, дар панели домени худ илова кунед.

## 🎨 Илова кардани иконка (Favicon):
Барои он ки сайти шумо нишонаи (logo) худро дошта бошад:
1. Сурати худро бо номи `icon.png` захира кунед.
2. Онро дар папкаи `src/app/` ҷойгир кунед.
3. Фармонҳои Git-ро (дар боло зикр шудаанд) иҷро кунед.
