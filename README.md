# KORYOB.TJ — Платформаи №1 барои дарёфти ҷойҳои корӣ дар Тоҷикистон

Ин лоиҳа бо истифода аз Next.js ва Firebase сохта шудааст. Шумо метавонед онро дар **Vercel** ё **Firebase App Hosting** ройгон сайт кунед.

## 🚀 Чӣ тавр сайт кардан (Deployment):

### Интихоби 1: Vercel (Осон ва ройгон)
1. Ба [vercel.com](https://vercel.com/) ворид шавед.
2. Тугмаи **"Add New"** ва баъд **"Project"**-ро пахш кунед.
3. Репозиторияи `bobojonov07/KORYOB-TJ`-ро аз GitHub пайваст кунед.
4. Дар бахши **Environment Variables**, ҳамаи калидҳои Firebase-ро аз файли `src/firebase/config.ts` илова кунед (ё танҳо калидҳои `.env`-ро).
5. Тугмаи **"Deploy"**-ро пахш кунед. Сайти шумо омода мешавад!

### Интихоби 2: Firebase App Hosting
1. Ба [console.firebase.google.com](https://console.firebase.google.com/) гузаред.
2. Ба бахши **App Hosting** гузаред ва GitHub-ро пайваст кунед.
3. Домени худро дар танзимот пайваст кунед.

## 🛠 Чӣ тавр ба GitHub бор кардан (Update):
Агар шумо дар код тағйирот ворид кунед, ин фармонҳоро дар терминал иҷро кунед:
```bash
git add .
git commit -m "Навсозии дизайн ва функсияҳо"
git push origin main
```
Сайти шумо (дар Vercel ё Firebase) ба таври худкор нав мешавад.

## 📋 Имкониятҳои барнома:
- **Ҷустуҷӯи кор**: Дар тамоми шаҳрҳои Тоҷикистон.
- **Чат дар вақти воқеӣ**: Мукотибаи мустақим байни корҷӯ ва корфармо.
- **Модераторӣ**: Филтри худкори дашномҳо ва блоккунии вайронкунандагон.
- **Писандидаҳо**: Имкони захира кардани эълонҳои ҷолиб.
- **Амният**: Ивази ном ва парол дар профил.
- **Дизайни муосир**: Мутобиқ барои телефон (PWA) ва компютер.
