# KORYOB.TJ — Платформаи №1 барои дарёфти ҷойҳои корӣ дар Тоҷикистон

Ин лоиҳа бо истифода аз Next.js ва Firebase сохта шудааст. Шумо метавонед онро дар **Vercel** ройгон сайт кунед.

## 🎨 Чӣ тавр иконкаи (Logo) сайтро илова кунед:
Барои он ки дар браузер нишони сайти шумо пайдо шавад:
1. Сурати худро ба компютер бор кунед.
2. Номи онро ба `icon.png` иваз кунед.
3. Ин файлро дар папкаи `src/app/` ҷойгир кунед.
4. Фармонҳои Git-ро (дар поён) иҷро кунед ва ба GitHub фиристед.

## 🚀 Чӣ тавр версияи навро ба GitHub бор кунед (Update):
Ин 3 фармонро паиҳам дар терминал иҷро кунед:

```bash
git add .
git commit -m "Update: Full-screen UI and delete functionality"
git push origin main
```

*Эзоҳ: Агар номи "main" кор накунад, "master"-ро санҷед.*

## ✨ Имкониятҳои нави ин версия:
- **Full-screen UI**: Экрани чат ва тафсилот дар мобилӣ пурра мебошанд.
- **Дизайни паймон (Compact)**: Header-и чат хурдтар шуд, то паёмҳо бештар намоён шаванд.
- **Ҳазфи чат ва паёмҳо**: Имконияти нест кардани таърихи суҳбат ва паёмҳои алоҳида.
- **Огоҳиномаҳо**: Нуқтаи норанҷӣ дар меню ҳангоми омадани паёми нав.
- **Вақти тоҷикӣ**: Намоиши вақти эълонҳо ба забони тоҷикӣ (масалан: "1 рӯз пеш").
