
# KORYOB.TJ — Платформаи №1 барои дарёфти ҷойҳои корӣ дар Тоҷикистон

Ин лоиҳа бо истифода аз Next.js ва Firebase сохта шудааст.

## 🚀 Чӣ тавр версияи навро ба GitHub бор кунед (Update):
Агар мушкилии "rejected" ё "divergent branches" барояд, ин фармонҳоро паиҳам иҷро кунед:

```bash
git add .
git commit -m "Update: Fix chat imports and logic"
git config pull.rebase false
git pull origin main
git push origin main
```

## 🎨 Иконкаи (Logo) сайт:
Файли иконкаи шумо дар `src/app/icon.png` ҷойгир аст ва ба таври худкор дар метадатаи сайт истифода мешавад.

## ✨ Имкониятҳои нави ин версия:
- **Чат ва Паёмҳо**: Ислоҳи мукаммали интиқоли паёмҳо ва огоҳиномаҳо.
- **WhatsApp Integration**: Тугмаи мустақим ба WhatsApp дар тафсилоти эълон.
- **Full-screen UI**: Экрани чат ва тафсилот дар мобилӣ пурра ва паймон мебошанд.
- **Ҳазф**: Имконияти нест кардани таърихи суҳбат ва паёмҳои алоҳида.
- **Амният**: Модератсияи автоматии паёмҳо ва блоккунии корбарони хатарнок.
