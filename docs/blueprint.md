# **App Name**: Salom Notes

## Core Features:

- Note Creation: Users can compose and save new notes with a simple text editor.
- Note Listing: Display a clean, scrollable list of all saved notes for easy browsing.
- Note Editing: Ability to select an existing note and modify its content.
- Note Deletion: Functionality to permanently remove unwanted notes.
- Greeting AI Assistant: An AI tool that can generate friendly greetings or positive messages based on a short user prompt.

## Style Guidelines:

- Inspired by welcoming warmth and clarity, a light color scheme is chosen. The primary color is a soft golden yellow (#E0B152). This provides a gentle, inviting presence. The background color is a heavily desaturated soft cream (#F5F3F0), ensuring high readability and a calm user experience. An accent color of rich terracotta orange (#B8452E) is used for calls to action and emphasis, offering an appealing contrast.
- The font for both headlines and body text will be 'Inter', a grotesque-style sans-serif. Its modern, objective, and highly legible characteristics ensure clarity and a neutral, professional appearance across the application.
- Simple, line-art style icons will be used throughout the application to maintain a clean and uncluttered interface, enhancing intuitiveness without visual overload.
- A clean, minimalistic layout with ample white space will provide a calming and focused user experience. Content will be well-organized and easily digestible on various screen sizes.
- Subtle, smooth transition animations will be implemented for navigation and interactive elements, offering refined visual feedback without distracting the user.
